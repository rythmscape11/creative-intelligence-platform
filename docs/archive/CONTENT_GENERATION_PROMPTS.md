# Content Generation Prompts for AI-Assisted Writing

This document provides prompts for generating high-quality, SEO-optimized content for all 30 marketing tool pages using AI assistants (ChatGPT, Claude, etc.).

---

## üéØ Master Prompt Template

Use this template for each tool to generate comprehensive content:

```
I need to create SEO-optimized content for a [TOOL_NAME] page on MediaPlanPro, a marketing tools platform.

**Tool Details:**
- Name: [TOOL_NAME]
- Category: [CATEGORY]
- Primary Keyword: [PRIMARY_KEYWORD]
- Target Audience: [TARGET_AUDIENCE]
- Main Benefit: [MAIN_BENEFIT]

**Content Requirements:**
1. Generate 20 FAQ items (80-120 words each) covering:
   - What is questions (definitions)
   - How to questions (step-by-step guides)
   - Best practices questions
   - Comparison questions
   - Troubleshooting questions
   - Advanced feature questions

2. Each FAQ should:
   - Include the primary or secondary keyword naturally
   - Provide specific, actionable information
   - Include numbers, statistics, or examples where relevant
   - Use bold for key terms
   - Be optimized for featured snippets

3. Write in a professional but approachable tone
4. Focus on practical, actionable advice
5. Include industry benchmarks and data points

Please generate the 20 FAQ items in this TypeScript format:

```typescript
{
  question: 'Question text?',
  answer: 'Detailed answer (80-120 words)...',
  keywords: ['keyword1', 'keyword2'],
},
```
```

## üìä Content Section Prompts

### How to Use Section (800-1,200 words)

```
Write a comprehensive "How to Use [TOOL_NAME]" section (800-1,200 words) that includes:

1. Introduction paragraph explaining the tool's purpose
2. 6-8 detailed steps with:
   - Step number and name
   - 150-200 word explanation
   - Specific examples
   - Pro tips in callout boxes
3. Visual descriptions of the interface
4. Common use cases
5. Expected results

Format with proper headings, bullet points, and callout boxes for tips.
```

### Benefits & Use Cases Section (1,000-1,500 words)

```
Write a "Benefits & Use Cases" section (1,000-1,500 words) for [TOOL_NAME] that includes:

1. Introduction paragraph
2. 4-6 key benefits with:
   - Benefit name
   - 100-150 word explanation
   - Specific metrics or data points
3. 4-6 real-world use cases with:
   - Scenario name
   - 150-200 word description
   - Specific example
   - Expected outcomes

Include industry-specific examples and quantifiable benefits.
```

### Best Practices Section (800-1,200 words)

```
Write a "Best Practices for [TOOL_NAME]" section (800-1,200 words) that includes:

1. Introduction paragraph
2. 6-8 best practices with:
   - Practice name
   - 100-150 word explanation
   - Do/Don't examples
   - Advanced tips

Focus on actionable, expert-level advice with specific examples.
